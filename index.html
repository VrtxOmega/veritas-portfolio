Architecture

⚡

Vanilla JavaScript

No frameworks, no build step, no bundler. Pure browser-native JS that runs instantly on any device.

🔐

WebCrypto SHA-256

Browser-native cryptography for tamper-proof credential hashing. No external crypto libraries.

💾

IndexedDB Storage

Client-side persistent storage for credentials. Works offline, no server required.

Credential Lifecycle

1

Challenge

User completes a cognitive skill challenge — timed, scored, validated.

2

Hash

Performance data is hashed using WebCrypto SHA-256 in the browser.

3

Sign

Credential is cryptographically signed and bound to the user's identity.

4

Store

Persisted to IndexedDB for offline access and future verification.

5

Verify

Third parties can verify credential integrity by re-hashing and comparing.

6

Share

Export as portable credential file or share via QR code for instant verification.
